# Screenshots Directory

This directory contains screenshots of the calendar application for documentation purposes.

## Required Screenshots:

1. **calendar-view.png** - Main calendar interface showing events
2. **event-form.png** - Event creation/editing form
3. **event-details.png** - Event details view

## How to Add Screenshots:

1. Take screenshots of your running application
2. Save them in this directory with the exact filenames mentioned above
3. Ensure screenshots are clear and show the key features
4. Recommended size: 1200x800 pixels or similar

## Screenshot Guidelines:

- Use a clean, populated calendar with sample events
- Show both the calendar view and form interactions
- Include different event types (single, recurring)
- Capture responsive design on different screen sizes if possible
